## Angle 1: Technical Deep-Dive

**Topic:** Persistent iframe tabs — no page reload when switching ecosystem views

**Why it's interesting:** Browser tabs typically unmount and remount components when you navigate. For embedded iframes (WaveWarZ, analytics dashboards), that means a full reload every time — terrible UX. We solved it with CSS visibility toggling: all iframes mount once, only the active one is `display: block`, the rest `display: none`. State persists. No flicker. No re-auth.

**Farcaster/X post (280 chars):**
> Small trick with big UX impact: instead of unmounting iframes on tab switch, we CSS-toggle visibility. All iframes mount once. Zero reload on switch. WaveWarZ battle stats stay live while you flip to Analytics. Sometimes "hide it" beats "remove it."

**Code angle:** `src/app/(auth)/ecosystem/` — persistent iframe tabs pattern
