## Categorized Findings

### P0 Bug: ZABAL Vote-Power Score (PR #592)
`haatz.quilibrium.com/v2/farcaster/user/bulk` returns HTTP 200 with `experimental: null`. The `fetchWithFailover` function interprets 200 as success — never falls through to Neynar. Every voter's `neynar_user_score` silently defaults to 0.5. Every vote multiplier has been wrong since the haatz integration shipped.

### P0 Bug: ZOE Archive Amnesia (PR #582)
ZOE appends every turn to a permanent on-disk JSONL archive (`~/.zao/zoe/archive/`). But `loadMemory()` never calls `readArchive()`. The archive is write-only from the context path's perspective. Every ZOE session can only see the last 8 turns. This explains every `[not in context]` complaint in the ZAO Civilization Telegram thread.

### Security: Pre-commit Hook Was Inert (PR #585)
The `.husky/pre-commit` hook file existed but `core.hooksPath` was pointing to `.git/hooks/` where nothing ran. The collision guard and secret scan both failed silently. A Telegram bot token and a test-fixture key leaked this month because of this gap. PR #585 wires it properly.
