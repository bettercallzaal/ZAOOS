## What Shipped

### ZOE Gets Smarter (6 features)

ZOE had the most active week of any subsystem. Six features landed:

1. **Append-only conversation archive** — ZOE now logs every conversation to a persistent file. Continuity across sessions, no drift.
2. **Grounding rules in persona** — ZOE now greps `research/` before answering questions about ZAO. Less hallucination, more accurate recall of the actual canon.
3. **Chunked Telegram replies** — Long responses were silently truncating at 4096 chars. Now properly chunked. Fixes a silent failure mode.
4. **Forward nudges from task queue** — Replaced hourly "tips" (generic, easily ignored) with nudges pulled directly from the task queue. ZOE now pings you about real work you said you wanted to do.
5. **Groups + elder/lineage + memory blocks** — ZOE can now distinguish group chats, recognize elder roles, and wire memory blocks per context.
6. **Fix: drop `--bare` for Claude OAuth** — ZOE's Claude invocation was failing auth silently. One flag fix, now working.

### Team-Bots: Magnetiq + AttaBotty

Two new conversational bots launched in a single PR (#503):

- **Magnetiq** — serves the Flow/blockchain events team. SAPS-aware, Whop-integrated. Reactive only (no nag crons).
- **AttaBotty** — serves William and DaNici's stream community. Hard rule: never volunteer William's real name.

Both bots share a single Node process, a shared Claude brain (Sonnet/Opus), and memory to Supabase. Post-launch audit (doc 645) came back GO SHIP — 9-item security checklist all PASS.

### Agent Canon Locked (doc 644)

The agent stack is now documented and frozen. 3 surfaces: ZOE, Hermes, ZAOstock bot. Hermes pattern as the standard for any future agent. Persona file layout standardized. Allowlist gates defined. No new bots without a research doc + Zaal approval. This is the operating system for the agent fleet going forward.

### zao-transcribe Script

New script (#523): give it a podcast or audio URL, get back a clean transcript. Built on Jina (URL fetch) + Whisper (transcription). Immediate use: BCZ YapZ podcast workflow. Longer-term use: ZOE auto-summarizing audio forwarded to the inbox (once AgentMail allowlist is fixed).

### SIDEQUESTZ Design Spec

New spec (#521): SIDEQUESTZ is a ZOE goal-alignment layer. You tell ZOE what you're working toward across a week or month. ZOE surfaces captures, nudges, and task carries in that frame. The nudge system is already live; SIDEQUESTZ gives it longer memory. Implementation lives in `bot/src/zoe/` — not a new bot, a ZOE enhancement.
