# ZAO Portal Infra

Version-controlled config for the ZAO phone-first agent portal running on VPS 1 (`31.97.148.88`). Canonical source of truth for:

- `ao.zaoos.com` — AO dashboard
- `claude.zaoos.com` — Claude Code TUI (ttyd + tmux)
- `portal.zaoos.com` — landing page + quick-spawn + todos + test checklist

## Quickstart on a new VPS

```
cd ~
git clone https://github.com/bettercallzaal/ZAOOS
cd ZAOOS/infra/portal
./install.sh
```

`install.sh` is idempotent. Safe to re-run after any sync.

## On an already-running VPS (normal flow)

```
cd ~/ZAOOS/infra/portal
./sync.sh
```

Pulls latest from git, refreshes symlinks, restarts `caddy`, `spawn-server`, `auth-server`, `ttyd` tmux sessions. Leaves `ao`, `claude-zaoos`, `cloudflared` untouched.

## Layout

```
caddy/
  Caddyfile              Port-based vhosts (no TLS -- tunnel terminates)
  portal/                portal.zaoos.com static (index.html, manifest, icon, todos.html)
  dock/                  dock.js (universal nav, copied into portal/ at install time)
  claude/                claude.zaoos.com wrapper (iframe to ttyd + dock)
  ao/                    ao.zaoos.com wrapper (iframe to AO :3001 + dock)
bin/
  auth-server.js         Cookie login (:3005) -- forward_auth target
  spawn-server.js        Spawn + todos + checklist backend (:3004)
  watchdog.sh            Respawns critical tmux sessions (cron every minute)
  start-agents.sh        @reboot bootstrap
  fix-node-pty.sh        Homebridge-fork symlink for node-pty (AO dashboard terminal stream)
  test-checklist-ping.sh Telegram nudge (cron every 15 min until all tests done)
cloudflared/
  config.yml             Tunnel ingress -> localhost ports
cron/
  crontab                Snapshot of `crontab -l` for reference
docs/
  (future runbooks)
```

## Ports

| Service | Port | Fronted by | Auth |
|---------|------|------------|------|
| AO dashboard | 3001 | Caddy :3002 wrapper | cookie |
| portal static + API | 3003 | Caddy | cookie |
| spawn-server backend | 3004 | Caddy | cookie |
| auth-server | 3005 | Caddy :3006 direct | none (login page) |
| ttyd | 7681 | Caddy :7682 wrapper | cookie |

## Secrets

- `~/.auth-token` — 64 char hex, used as cookie value. NOT in git. Generated by `install.sh`.
- `qwerty1` login password is hardcoded in `bin/auth-server.js` for now. **Rotate via env var in a future pass** (move to `~/.env.portal`).
- Cloudflared credentials at `~/.cloudflared/b5025a71-37d1-4579-9bed-9aa2f17e712e.json`. NOT in git.

## Surfaces + URLs

- `https://portal.zaoos.com/` — main portal (quick spawn + history + templates + tiles + dock)
- `https://portal.zaoos.com/todos` — todo list (prioritize, filter, bulk add, spawn-from-todo)
- `https://portal.zaoos.com/test-checklist` — test tracker
- `https://ao.zaoos.com/` — AO dashboard via iframe wrapper with dock
- `https://claude.zaoos.com/` — Claude Code via iframe wrapper with dock
- Backend: `/spawn-action`, `/api/spawn`, `/todos`, `/todos/<id>`, `/test-done?id=N`
