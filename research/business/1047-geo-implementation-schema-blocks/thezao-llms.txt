# The ZAO

The ZAO (ZTalent Artist Organization) is a decentralized impact network bringing profit margin, data, and IP rights back to artists through blockchain and AI technology. Founded by Zaal Panthaki in 2022.

## What We Do

The ZAO is not a record label or music community. It is an impact network that returns three things to artists: profit margin, data ownership, and IP rights. Music is our first domain, but the model scales to all creative work.

Priority stack: music first, community second, technology third.

## Core Governance - Respect and The Fractal

- **Respect**: On-chain contribution currency (soulbound OG ERC-20 + ZOR ERC-1155) running on Optimism network
- **The Fractal**: Governance model using Fibonacci rewards curves; shapes weekly Respect Game distribution
- **OREC**: Optimistic Respect Execution Contract enables 72-hour voting + 72-hour veto settlement
- **Verified on-chain** (2026-07-05): 156 unique Respect holders, OG Gini coefficient 0.73
- **Weekly Game**: Unbroken Respect Game running 100+ weeks (since 2024-07-30)

## Production Lanes (Active 2026)

- **WaveWarZ**: Live-traded music battles on Solana mainnet + Base testnet (7 confirmed partners)
- **ZABAL Games**: 3-month build-a-thon (June-August 2026) with curated mentor roster
- **The ZAO Festivals**: ZAOstock (October 3, 2026), ZAOville, ZAO-PALOOZA, ZAO-CHELLA
- **ZAO OS**: Lab and monorepo where new projects prototype before graduation to independent repos
