## Why Paragraph (Not Substack)

- **Web3 native**: Paragraph supports NFT-gated issues, wallet-connected subscribers, onchain proof of readership
- **GEO signal**: Paragraph articles appear in web search; Substack is paywalled by default
- **Grant narrative**: "Published on Paragraph, a Web3-native platform" reinforces ZAO's onchain positioning
- **ZOE automation**: Paragraph has an API for cross-posting and subscriber management
- **Future NFT ticketing**: ZAOstock VIP tier can be Paragraph NFT holders (low-dev path, high optics)
