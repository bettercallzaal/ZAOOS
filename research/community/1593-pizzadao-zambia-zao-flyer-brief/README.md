# 1593 — ZAO x PizzaDAO Zambia: Flyer Brief

> **What this is:** Copy brief + visual spec for a ZAO flyer for PizzaDAO Zambia. Board task: "ZAO flyer for PizzaDAO Zambia." From doc 670, Iman is the connection between ZAO and the Zambia chapter. The flyer promotes ZAO/WaveWarZ to the PizzaDAO Zambia community — audiences overlap on Web3 music, Africa, and community events.

---

## Context: Why PizzaDAO Zambia

PizzaDAO is a worldwide Web3 community that runs pizza events in every major city. The Zambia chapter is one of the most active African chapters. Iman has direct connections there (doc 670).

**The ZAO × Zambia opportunity:**
1. Africa Battle Week (Sep 26) needs a West African artist — PizzaDAO Zambia may know WaveWarZ-compatible artists
2. ZAO's "loser-earns" philosophy aligns with PizzaDAO's "everyone should be paid" community ethos
3. The flyer converts PizzaDAO Zambia attendees into ZOR holders and /wavewarz followers

## Visual Design Spec

### Both Versions

| Element | Spec |
|---------|------|
| Size | 1080×1080 (Instagram square) + 1080×1920 (Story/WhatsApp) |
| Color palette | ZAO primary black (#000000) + gold (#FFD700) + white text. Accent: Zambia green (#198A00) for the ZAO × Zambia collab |
| Font: Headline | Bold, modern sans-serif (Montserrat Bold or equivalent) |
| Font: Body | Montserrat Regular or Open Sans |
| ZAO logo | Top-left or top-center. Transparent PNG |
| PizzaDAO Zambia logo | Bottom-right or co-badge. "In partnership with PizzaDAO Zambia" |
| QR code | Link to thezao.xyz/wavewarz (Version A) or ZAOstock Eventbrite link (Version B) |
| Background | Dark gradient (black to deep green) or abstract soundwave pattern |

### ZAO Brand Elements to Include

- "The ZAO" text logo
- "WaveWarZ" wordmark or logo (from wavewarz.info)
- Battle-energy visual: two mics, soundwaves, stage lighting

**Don't include:**
- Crypto prices or token values (off-brand)
- Lengthy legal text
- Multiple CTAs — one per version
