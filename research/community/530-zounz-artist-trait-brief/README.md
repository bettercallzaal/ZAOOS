# 530 - ZOUNZ Artist Trait + Wordmark Brief

> **Goal:** Deep wealth-of-info brief for artist designing ZOUNZ NFT traits + ZOUNZ wordmark/font. Wraps ZAO ethos, visual identity, trait categories, lore, font direction, and Nouns Builder technical spec into one drop.

ZOUNZ = ZABAL Nouns DAO on Base chain (`0xCB80Ef04DA68667c9a4450013BDD69269842c883`). Daily NFT auction, treasury funds music releases, holders vote on what The ZAO distributes. Like Nouns but for an indie music community born in Ellsworth, Maine.

| Decision | Recommendation |
|----------|----------------|
| **Aesthetic anchor** | USE indie label visual DNA: Sub Pop early 90s zine flyers, Stones Throw artist-first chunkiness, Numero Group archival sparseness. NOT corporate web3. |

## Section 1 - What The ZAO Is

The ZAO = ZTalent Artist Organization. Decentralized music community on Base. ~188 members. Artists keep their revenue, curators earn reputation, community governs itself.

**Positioning order (do not break this):** music first, community second, technology third. We do not lead with crypto. Show, do not explain web3. Source: research/community/432-zao-master-context-tricky-buddha.

**Flagship event:** ZAOstock - Oct 3 2026, Franklin St Parklet, Ellsworth Maine. Co-produced with Art of Ellsworth.

**HQ neighborhood:** Ellsworth, Maine. Steve Peer (Crown Vics drummer, ex-CBGB roadie for Richard Hell) has been running 430 Bayside house concerts since 1989. ZAO acquired house in Ellsworth as annual retreat HQ. Maine is foundational, not scenery.

## Section 2 - Visual Aesthetic + Moodboard

### Concrete visual references (study these)

- **Sub Pop early 90s** - punk ethos hiding premium production. Worn screen-printed signage, photocopied zines, hand-drawn type on concert flyers. Lo-fi wrapper, premium inside.
- **Numero Group reissue covers** - meticulous typography, sparse white space on deep backgrounds, archival sensibility.
- **CBGB / NYC punk venue culture** - Steve Peer roadied for Richard Hell at CBGB, brought "music in the room with you" philosophy to Ellsworth. Bold hand-lettering, limited palettes, raw printing texture.
- **Stones Throw Records** - artist-first label, chunky type with intentional roughness, organic geometry.
- **Drag City** - radical simplicity, lowercase-first, minimal flourish.

### Texture + material palette

- Cassette J-cards (thick cardstock, screen-printed, worn edges)
- Zine cut-and-paste (visible crop marks, hand-drawn annotations, inconsistent type sizes)
- Screen-printed posters (heavy ink, color registration drift, halftone dots on cream/yellow)
- Sharpie on kraft (paper-textured backgrounds, bold black weight)
- Lobster trap rope + driftwood (weathered gray, frayed edges, salt-stained)
- Letterpress impressions (ink-pressure variation, pressed-official quality)

**Avoid:** glossy flat digital surfaces, perfect anti-aliasing, smooth gradients.

### Color usage in the wild

| Color | Hex | Use |
|-------|-----|-----|
| Background navy | `#0a1628` | Canvas, every page bg, never pure black |
| Surface navy | `#0d1b2a` | Cards, panels |
| Elevated navy | `#1a2a3a` | Modals, dropdowns |
| Primary gold | `#f5a623` | Interactive only - CTA, hover, active tab, badge |
| Hover gold | `#ffd700` | Brighter gold + shadow glow |
| Off-white | `#ededed` | Body text, never pure white |
| Muted gray | `#a0aec0` | Metadata, timestamps |

**Rule:** gold means action. A trait labeled "Gold Member" should not be all-gold; let navy carry weight, gold accent one edge.

### Maine + Ellsworth visual hooks

- Franklin Street Parklet (literal ZAOstock stage)
- Granite + fog overlay (no cartoon lighthouse)
- Lobster trap rope knots (working culture, not tourist clipart)
- Pine + fall foliage (Oct 3 = burnt sienna, goldenrod, deep forest)
- Weathered wood storefronts, hand-painted business signs
- Salt air oxidation on metal, blueberry stain

### What ZAO is NOT visually

- Corporate web3 (no laser eyes, rockets, moons, gradient backgrounds)
- NFT NYC convention booth aesthetic
- Polished startup branding (no Helvetica Neue, no 12-column grids)
- Generic crypto iconography
- Emojis, em dashes - Zaal will reject these instantly

## Section 4 - Wordmark + Font Direction

### Reference wordmarks to study

- **Nouns DAO "NOUNS"** - direct ancestor, blocky high-contrast pixel-adjacent letterforms
- **Stones Throw Records** - chunky serif with intentional roughness, organic geometry
- **Sub Pop / Matador** - 90s indie label, slightly condensed, defiant lowercase
- **Drag City** - radical simplicity, lowercase-first since 1990
- **Numero Group** - archival/catalog mark feel, every letter has weight

### Type style direction

| Dimension | Direction |
|-----------|-----------|
| Weight | Chunky 600-900. Music doesn't whisper. |
| Geometry | Geometric sans with character. Not Swiss cold, not graffiti chaos. Mid-ground. |
| Case | All-caps. ZOUNZ works better than Zounz or zounz. |
| Serif | Sans core, optional thin serif accent on key letters (Z) |
| Hand-drawn | Digital execution with hand-drawn FEELING. Subtle axis variation, tighter tracking. |
| Distress | Wordmark stays clean (works at 32x32). Texture overlay (halftone, paper grain) for poster only. |

### Letter shape ideas for Z-O-U-N-Z (3 Zs as signature motif)

**Option A - Mirror Z motif (RECOMMENDED).** First and last Z mirror each other, center O is the pivot. Visual palindrome. Recognizable rotated, mirrored, overlapped. Echoes cassette symmetry (tape reels), blockchain symmetry (verification), and verse-chorus-verse song structure.

**Option B - Z-as-soundwave.** Diagonal strokes of Z get +/-2-3 degree subtle curve suggesting audio vibration. Ultra-thin soundwave threads between letters.

**Option C - Z-as-cassette-spool.** Outer Zs get small circular elements at top/bottom (tape spool circles). Analog cassette meets digital on-chain.

**Option D - Z-as-numbered-index.** Each Z carries small superscript number (Z1, Z2, Z3). Ties to Nouns Builder where each trait is numbered.

**RECOMMEND:** Mirror Z (Option A) as core wordmark. Soundwave (B) as audio-context motion variant. Spool (C) as poster/merch texture layer.

### Font pairing stack

| Use | Font | Why |
|-----|------|-----|
| Wordmark | Custom geometric sans (built for ZOUNZ) | Mirror-Z motif as identity element |
| Marketing headlines | Inter ExtraBold 700-900 | Already in ZAO OS globals.css |
| Marketing body | Inter Regular 400-500 | Already system font |
| Trait labels small text | Departure Mono or IBM Plex Mono | Mono grid echoes on-chain pixel grid |
| Numbers / dates | JetBrains Mono Medium | Numbers read perfectly aligned |
| Retro Easter egg overlays | Pixelify Sans, Press Start 2P | Sparingly, posters only |

All open-source - OFL/Apache/MIT. Grab from Google Fonts or GitHub.

### Small-size legibility (32x32 trait grid)

- 16px ultra-small: letters only, no serifs, uniform weight
- 24px small: ZOUNZ with optional spool circles
- 40px+ normal: full wordmark with detailing + texture overlay option
- Trait number stamp: Departure Mono 8-10px, all caps "TRAIT #1847"
- Auction countdown: JetBrains Mono Bold 11px, "2d 4h 17m"
- Artist credit: Inter Semibold 9px, "@stilo_world"

## Section 5 - Nouns Builder Technical Spec

### Canvas + format

- **Format:** PNG (raster, recommended) or SVG (vector)
- **PNG dimensions:** 600x600 minimum, 1024x1024 recommended for high-DPI
- **SVG:** 32x32 minimum, scale-invariant
- **Square only.** No rectangles.
- **Crisp pixel edges.** No anti-aliasing, no blur, no soft shadows.

### Layer order (Z-stack bottom to top)

1. Background (lowest)
2. Body
3. Accessory
4. Head
5. Glasses (highest)

Reorderable in Nouns Builder UI before finalize.

### Transparency rules

- All trait PNGs must be 32-bit PNG with alpha channel
- Pixel-perfect transparent backgrounds
- No anti-aliased edges - matches Nouns aesthetic
- Each layer composites separately

### Color palette discipline

- Original Nouns use DawnBringer 32-color palette (lospec.com/palette-list/dawnbringer-32)
- Recommend 256 colors max across all traits (PNG 8-bit) for file size
- Nouns Builder does NOT enforce shared palette - each PNG can use own color space
- Optional: deliver shared palette JSON for cross-trait color harmony reference

### Upload mechanism

```
ZOUNZ_Traits/
  background/
    bg-mint.png
    bg-navy.png
  body/
    body-slim.png
  accessory/
    acc-chain.png
  head/
    head-round.png
  glasses/
    glasses-dark.png
```

- Max 200 MB total
- Upload via [Nouns Builder UI](https://nouns.build/dao/base/0xCB80Ef04DA68667c9a4450013BDD69269842c883)
- Reorder layers if needed in interface
- Images stored on IPFS, on-chain MetadataRenderer generates random combos at mint

### Naming convention

`[property]-[variant]-[descriptor].png`
- Lowercase, hyphens only
- No spaces, underscores, camelCase, special chars
- 64 char max per filename
- Property folder name (not filename) determines trait type

### Preview rendering

- Local: composite PNGs manually in Procreate / Aseprite / Photoshop to verify stacking
- Screenshots at 320x320 (small) and 1024x1024 (detail)
- On-chain: MetadataRenderer auto-generates random combos at mint via `keccak256(blockhash, tokenId)`
- **Test on Sepolia testnet first** before mainnet trait deploy
