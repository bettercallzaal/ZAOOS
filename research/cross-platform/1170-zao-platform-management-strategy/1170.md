# ZAO Platform Management Strategy: One System, All Platforms

## Key Decisions

| Decision | Answer |
|----------|--------|
| **Content origin** | Ship → clip → schedule. ZAO distributes what it builds; it doesn't manufacture content for content's sake |
| **Primary posting tool** | Firefly (Farcaster + X simultaneous) for text; Postiz for scheduled video clips; Restream for live simulcast |
| **Clip engine** | zaalclip (Livepeer) → FlowStage aesthetic overlay (see Doc 1169) → Postiz schedule |
| **Clipping accounts** | One clip-focused account per platform (prep'd below); Zaal creates accounts + pastes API keys to activate |
| **Voice anchor** | Every post, every platform: open with ZM. No emojis. No hashtags. Specific numbers, not "many" |
| **Handle priority** | Fix X display name + WaveWarZ YouTube name first (blocks GEO/LLM identity); @wavewarz IG squatted — needs decision |

## The One-System Pipeline

```
CONTENT SOURCE
│
├─ Live show (COC Concertz, WaveWarZ battle)
│    └─ Restream → simulcast to YouTube, TikTok, X, Twitch (live)
│         └─ zaalclip → Livepeer clip segment (post-show)
│
├─ WaveWarZ battle audio / COC DJ set audio
│    └─ FlowStage → aesthetic short-form video (30–60s render)
│
├─ Written updates (ZIPs, governance, ship announcements)
│    └─ Firefly → Farcaster + X (single keystroke cross-post)
│
└─ Long-form (YouTube main channel, LinkedIn articles)
     └─ Manual / scheduled via Postiz
              │
              ▼
         CLIPPING ACCOUNTS (per platform — see below)
              │
              ▼
    Postiz schedules + publishes
    Bonfire episode per clip batch (knowledge graph trail)
```

## Tooling Stack

| Tool | Role | Auth/Access | Cost |
|------|------|-------------|------|
| **Firefly** | Farcaster + X cross-post | Native app | Free |
| **Postiz** | Video clip scheduling, all platforms | `POSTIZ_API_KEY` in zaalclip env | Self-hosted (open-source) |
| **Restream** | Live simulcast | Restream account | Paid tier |
| **zaalclip** | Livepeer clip extraction | Livepeer API + Cloudflare | ZAO infra |
| **FlowStage** | Short-form video rendering from audio | OAuth popup → `fs_` key (any plan) | Free tier available |
| **Bonfire** | Knowledge graph — one episode per clip batch | `BONFIRE_API_KEY` | ZAO infra |

### Tooling decision rationale

- **Firefly for text posts**: Zaal's existing workflow. One post hits Farcaster (the onchain-native community) and X (broader reach). Don't break what works.
- **Postiz for clips**: Open-source, self-hosted, already has `POSTIZ_API_KEY` in zaalclip. Supports TikTok, Instagram, YouTube, LinkedIn, Facebook, X, Pinterest in one schedule. No per-post fees.
- **FlowStage for audio-to-clip**: Bridges WaveWarZ battle audio → short-form Reels/TikTok without manual video editing. Pairs with Postiz.
- **Restream for live**: Multi-destination live (YouTube + TikTok + X Live simultaneously). Separate concern from clip scheduling.
