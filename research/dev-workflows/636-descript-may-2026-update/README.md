## Features

### Underlord Model Picker (Claude Sonnet 4.5)

Underlord now lets you choose your AI model. Swap between versions at any time - pick quality, speed, or cost tradeoff. Every LLM request in the agent respects your choice. Shipped May 2026.

### Transcription Admin Controls

New admin-level settings for:
- Transcription glossary (brand terms, names)
- Do-not-translate lists (names, brand phrases)
- Audio/video preview BEFORE Underlord applies changes

### Transitions & Scene Transitions

Apply effects directly via transcript selection:
- Fades, crossfades, wipes, star wipes, blur
- Animations: Ken Burns, pans, zooms
- No clip management needed
- Works with reusable branded templates

### Media Library Upgrade

Upload + organize media at drive level. Reuse across projects. Saves transcription + media minutes.
