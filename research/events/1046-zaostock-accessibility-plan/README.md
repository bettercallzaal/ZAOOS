# 1046 — ZAOstock Accessibility Plan: A Real, Honest Baseline Under $3K

> **Goal:** Doc 1032 covers one accessibility item - ADA-compliant portable restrooms. Nothing else has ever been planned: mobility access, sensory accommodations, accessible parking, ASL interpretation, or even a public accessibility statement. This doc builds a realistic, honestly-scoped plan that fits inside doc 1013's existing budget.

## A legal-characterization caveat before the findings

This doc's research treated ZAOstock as subject to ADA Title II (the public-entity standard) on the reasoning that Franklin Street Parklet is city-adjacent space and the event involves city permitting. That's a reasonable read, but it isn't confirmed: ZAOstock itself is privately organized (via fiscal sponsorship, per doc 1045) and the venue is rented through Heart of Ellsworth, a private nonprofit - not a direct city-run event. Whether Title II (public entity) or Title III (public accommodation, the standard for privately-run public events) actually applies is a real legal question this session's research couldn't settle. Practically, it barely matters for what to actually do - the baseline accommodations below are good practice under either standard - but the exact legal obligation level should be confirmed, not assumed, before treating any of this as a compliance requirement rather than a strong recommendation.

## Key Decisions

| Recommendation | Why |
|---|---|
| **Implement the baseline now: accessible routes, restroom on a firm surface, accessible parking/drop-off, a public accessibility statement, and a service-animal policy** | These five items are inexpensive (roughly $500-$1,000 combined), achievable with doc 1032's existing volunteer structure, and represent genuine best practice regardless of exactly which legal standard applies |
| **The ADA portable restroom must sit on a firm surface, not grass - this is a real gap in the current plan** | Doc 1032's plan already specifies one ADA unit per porta-potty cluster, but never addressed the ground surface underneath. A wheelchair user cannot use an ADA-compliant toilet if the approach is soft grass or mud - this is a cheap fix ($250-$400 for platform/matting) that closes a real functional gap, not just a paperwork one |
| **Offer ASL interpretation on advance request, not as a guaranteed default** | This is standard practice even at well-funded festivals (Britt Music & Arts Festival, ACL Festival, Shaky Knees all use a request-based model). It keeps the budget flexible ($0 unless a request comes in, then $700-$900) while still making the accommodation genuinely available |
| **Use the existing First Aid tent as a dual-purpose quiet/sensory space, not a separate rental** | Zero additional cost, and doc 1032's First Aid station already has the physical presence and staffing this needs. This is exactly the kind of "achievable within the existing plan" accommodation worth prioritizing over one that needs new budget or space |
| **Publish an honest accessibility statement now, distinguishing what's guaranteed from what's request-based** | Every well-regarded example found (Lake George Music Festival, Peninsula Music Festival) is explicit about this split. Overpromising and then failing to deliver on-site is worse for a disabled attendee than a clear, honest statement of real limits |

## Findings

### 1. What a genuinely achievable baseline costs: roughly $1,250-$2,800

Sourced research converges on a realistic total for direct accessibility costs at this scale: firm surface under the ADA restroom ($250-$400), parking/wayfinding signage ($100-$300), an accessible viewing area setup near the stage ($100), a stage ramp if the stage is raised more than ~6 inches ($800-$1,500 - genuinely required, not optional, if the stage has a meaningful riser), and a large-print program option if one is printed ($100-$200). ASL interpretation stays at $0 unless a request actually comes in, then $700-$1,200. All of this fits comfortably inside doc 1013's $2,000 contingency line without needing a new budget category. [FULL - per subagent research, cross-referenced against doc 1013's existing budget structure]

### 2. The restroom-surface gap is real and specific to ZAOstock's current plan

Doc 1032 already plans one ADA-compliant portable toilet per cluster (2 clusters total) but never specifies the ground surface. An ADA-compliant unit placed on grass, mud, or loose gravel doesn't function as accessible in practice - a wheelchair can't approach it. This is a genuinely new, specific finding: the fix is cheap (a plywood platform or rubber matting, $250-$400 total for both clusters) but it has to be ordered alongside the toilet rental, not assumed to come with it. [FULL - per subagent research]

### 3. Stage ramp: check the actual stage height before assuming it's needed or not needed

If ZAOstock's stage (per doc 1030/1032's planning, sourced from Wallace Events per the confirmed-facts memory) is raised more than roughly 6 inches, a compliant ramp is a real requirement, not optional - typical modular ramp rentals for a small festival stage run $800-$1,500 depending on height and supplier. If the stage is ground-level or has only a minimal riser, this cost may not apply at all. Nobody has checked the actual stage height against this question yet - it's a five-minute confirmation with a real budget swing attached. [FULL - per subagent research; ZAOstock's specific stage height not yet confirmed against this finding]

### 4. Accessible viewing doesn't need to be elaborate at this scale

For a 500-person, single-stage, ground-level festival, a raised viewing platform isn't necessary - a roped-off, clearly signed, ground-level area near the stage with a handful of chairs is what comparable small festivals actually do, and it costs about $100 in materials and rope/signage. This is one of the cheapest, highest-value items on the list. [FULL - per subagent research]

### 5. Real examples show the honest-statement pattern is standard, not exceptional

Lake George Music Festival (comparable ~500-person scale) and Peninsula Music Festival both publish accessibility statements that explicitly separate guaranteed accommodations (parking, restrooms, seating) from request-based ones (ASL, large print) with a clear contact and deadline. Even the Smithsonian Folklife Festival - vastly larger and better-funded - uses the same guaranteed-vs-request-based structure for some services. This confirms the honest-scoping approach in this doc's Key Decisions isn't a compromise - it's how good accessibility planning actually works at any scale. [FULL - per subagent research, real named festival examples]
