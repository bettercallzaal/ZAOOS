# 1063 - DevCon America: a US-builder community-event route into Devcon 8

> **Goal:** Find the mechanism (not yet used by anyone in the ZAO network) that lets a US-based ZAO cohort - separate from Zaal's own India trip in Doc 945 - earn funded Devcon 8 tickets and legitimacy, closing the gap between heavy India-side energy (Doc 945/954, Road to Devcon Wave 2 open) and near-zero US-side organizing energy.

## Key Decisions

| # | Decision | Rationale |
|---|----------|-----------|
| 1 | TARGET the Ecosystem Support Program (ESP) directly at esp.ethereum.foundation, NOT the "Road to Devcon" page | devcon.org/en/road-to-devcon/ is explicitly India-branded - every listed event is in an Indian city (Mumbai, Delhi, Bengaluru, Kerala, Jaipur), building toward Devcon 8 India. But the Ecosystem Program page (devcon.org/en/ecosystem-program/) states NO geographic restriction in its own eligibility text - it funds "meetups, bootcamps, hackathons, roundtables, academic collaborations" anywhere, it just happens to be promoted through the India lens because that is where Devcon 8 physically sits. |
| 2 | HOST one qualifying event inside ZAO's existing US network, not a new invented audience | ZAO already has the raw material for a real event: Ethereum Build Camp graduates (Doc 945, demo day July 11), ZABAL Games track builders (Artist/Builder/Creator, running through end of August), and the BCZ YapZ podcast audience. A single roundtable or small meetup pulling these three together is a genuine "hands-on participation" event under ESP's own criteria - not a fabricated qualifier. |
| 3 | APPLY before the verified August 14, 2026 ESP deadline | Confirmed live on esp.ethereum.foundation via devcon.org/en/ecosystem-program/ (fetched 2026-07-13). Response SLA is 15 days, so applying by early August still clears a pre-Devcon answer. |
| 4 | TREAT this as a second, parallel lever - not a replacement for Doc 945 | Zaal's own India trip is already funded through the Builder Discount / Scholars / crowdfund stack in Doc 945. This doc is about unlocking tickets and standing for OTHER US-based builders in the ZAO network who have no logistics plan today. |
| 5 | DO NOT commit to one grant-dollar figure before applying | Three different sources give three different numbers (see Findings). Confirm the live figure in the ESP application form itself before it goes into any pitch or budget. |
| 6 | IF the US cohort wants a standing presence (not just a one-time grant), file a DIP on forum.devcon.org | Same mechanism Doc 954 already flagged for the Community Hub route (RFP-13) and the Music Stage (Doc 700, Fellowship of Ethereum Musicians precedent) - DIPs are how community groups get programming space at Devcon, regardless of the applicant's home country. |

## Findings: the four Devcon entry mechanisms, compared

| Mechanism | Geographic framing | What it gets you | Deadline (verified) | Effort to qualify |
|---|---|---|---|---|
| Road to Devcon | India-branded; every listed event is in an Indian city | Visibility inside the India-specific campaign; not itself a funding instrument | Rolling waves through Nov 2026 (Wave 2 open per India-track log) | Low - mostly a listing/amplification page |
| Ecosystem Support Program (ESP) | No stated geographic restriction in its own criteria | Cash grant (reported $300-$1,000 depending on source, see below) + up to 5 free/discounted Devcon tickets, based on contribution and availability | August 14, 2026 (esp.ethereum.foundation) | Medium - must run a real meetup/bootcamp/hackathon/roundtable |
| Community Hub (RFP-13) | Global submission process, physically hosted in Mumbai | A staffed 4-day educational space inside Devcon 8 itself | Aug 12 submission, selections Aug 26 (Doc 954) | High - 2-3 co-organizers, full-conference staffing |
| DIP / forum.devcon.org | Global, text-first proposal | Community programming slot (precedent: music stage via Fellowship of Ethereum Musicians, Doc 700) | Rolling, community-discussed | Low-medium - a forum post plus follow-through |

**The gap this closes:** Doc 945 and Doc 954 already mapped every one of these levers for Zaal's own India trip. None of that prior work asked "does this only work if you're already in India?" - the Road to Devcon page reads India-first because that's the physical event location, but the ESP mechanism underneath it is not gated by applicant location. Nobody in the ZAO network has tested that reading yet.

## The grant-amount contradiction (do not paper over this)

| Source | Figure cited | Date checked |
|---|---|---|
| devcon.org/en/ecosystem-program/ (fetched direct) | Up to $300 USD + up to 5 tickets | 2026-07-13 |
| Doc 954 (this library, live-checked 2026-07-03) | Up to $500 grant + up to 5 tickets | 2026-07-03 |
| General program description surfaced via web search | Up to $1,000 for small-scale events | 2026-07-13 |
| Devcon 8 India Ecosystem Program page copy | "Up to $300 USD in financial assistance" | 2026-07-13 |

Three live sources, three numbers, all within the same two-week window. Do not put a specific dollar figure in front of a US cohort or a pitch deck - state "a cash grant plus up to 5 tickets, confirmed at application" until the ESP form itself returns a number.

## What a qualifying US event could actually look like

Built from material that already exists, not invented programming:

- A single virtual roundtable pairing Ethereum Build Camp graduates (Doc 945, demo day July 11, 228 registered camp-wide) with ZABAL Games Builder-track submitters (build-a-thon runs through end of August) - "shipping onchain, six weeks out from Devcon."
- Recorded and distributed the way ZAO already distributes: a BCZ YapZ episode, a Farcaster/X recap, a newsletter mention (400+ editions per Doc 945's proof numbers).
- Framed to ESP as a hands-on knowledge-sharing session under Ethereum's CROPS criteria (Censorship Resistance, Open Source, Privacy, Security) - the ZABAL Games Builder track and Empire Builder/Clanker/POIDH tool-chain already sit inside that frame.

This is deliberately the smallest real thing that qualifies - not a proposal for a new conference or a new team.
