## Findings

### 1. Portable toilets - a real local Ellsworth vendor exists

Ray Plumbing & Heating, 180 Abram Way, Ellsworth ME 04605, (207) 667-5536, Monday-Friday 8am-4:30pm.
$140/weekend standard unit, $235/weekend ADA-accessible unit, same-day delivery available. Regional Maine
pricing runs $90-250/day depending on provider and ADA-compliance. Other Ellsworth-area options if Ray
Plumbing can't cover the date: Casella (207-561-6611), Ellsworth Porta Potty Rentals (207-477-5060). Book
as soon as the date is locked - October is moderate season, not peak, so availability risk is low but not
zero. [FULL - 5 vendor sources checked, local + regional]

### 2. Radios - the cheapest of the three, was never a real budget concern

Dollar Radio Rentals ships nationwide at $1.99/day/unit with an on-time delivery guarantee - 8 radios for
one day runs about $16 plus shipping. EventTone Wireless is a Maine-based alternative (855-396-5450) with
free delivery and fast quote turnaround if a local contact is preferred over a national ship-in vendor.
Doc 1030's earlier $250-300 line item for "radio rental (if budgeted)" was an unpriced placeholder - real
pricing is an order of magnitude lower. [FULL - 5 vendor sources checked]

### 3. First aid - buy, don't rent, given the event cadence

A Red Cross 25-Person First Aid Kit (118-piece, OSHA-compliant) runs $50-100 to buy directly from
redcross.org/store - bandages, gauze, trauma pads, antiseptic, cold packs, gloves, a splint, and a first
aid guide. Renting a comparable kit runs about $20/day through event-supply companies like Executive CPR &
First Aid, which only wins if there's no reuse case - but ZAOville (Jul 25) happens before ZAOstock (Oct
3), so a purchased kit gets used twice, not once. An AED rental is a separate, optional add-on (~$99-135
for a one-day rental including delivery via CPR1, all-50-states service) - not assumed necessary for this
event size, but worth a look if the budget stretches. [FULL - 4 vendor sources checked]
