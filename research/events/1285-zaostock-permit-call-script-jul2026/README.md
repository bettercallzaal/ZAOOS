- Get the contact's name again

## What to Do After the Call

1. **Email follow-up within 1 hour** — summarize what was discussed and ask them to confirm. This creates a paper trail.
2. **If license IS required:** Start the surety bond process immediately. Bond providers in Maine include Merchants Bonding, SurePath, and others. Apply by Aug 10 for safety (5-7 day processing, and the bond must be submitted with the permit application before Aug 19).
3. **If EMS standby IS required:** Add $1,500-2,000 to the ZAOstock budget and update the bank sponsor pitches (doc 1277) to include this line item.
4. **Board task update:** Mark f0b22571 or the relevant ZAOstock permit board task as DONE + add the answers as notes.
