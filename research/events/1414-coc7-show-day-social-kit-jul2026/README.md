## Battle Control Commands

During the show, use these to manage battles:

```bash
# Create a new battle (closes any live battle first)
npx tsx scripts/manage-battle.ts create "Battle Title" "Artist A" "Artist B"

# Check live vote counts
npx tsx scripts/manage-battle.ts status

# Close the battle and tally winner
npx tsx scripts/manage-battle.ts close
```

Trigger the T+30 social posts WHEN you run `create`, not before.

## Off-Voice Flags

These words/patterns are banned from COC voice (per concertz.config.ts):
- "excited to announce"
- "leverage" / "synergy" / "game-changer"
- Em-dashes (use commas or new lines instead)
- Emojis in X posts (Farcaster: sparingly OK)
