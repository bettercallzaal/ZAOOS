## What the ZAO Presence at DevCon Could Look Like

### Option A — Side Event: ZAO Music Night (Recommended)

**Format:** A dedicated ZAO/WaveWarZ side event during the DevCon week (not at the main DevCon venue — too hard to get stage time). DevCon week has dozens of side events; ZAO could run its own.

**Venue:** A venue in BKC (Bandra Kurla Complex) or nearby, capacity 50-100. Venue cost in Mumbai: $200-500 for a small studio/bar with PA system.

**Program:**
- 7:00 PM: Doors open
- 7:30 PM: WaveWarZ intro (5 min — what is it, how does it work)
- 7:40 PM: Battle 1 — two pre-selected artists, live vote via WaveWarZ
- 8:00 PM: 20-min musical set / DJ between battles
- 8:20 PM: Battle 2
- 8:40 PM: Live Q&A: "Running a music DAO on Optimism for 100+ weeks"
- 9:00 PM: Open networking

**Artists:** Priority — Mumbai/India-based musicians for cultural relevance. Zaal scouts via Instagram DMs in September. Alternative: ZAO community artists who are traveling to DevCon.

**Cost:** $500-800 venue + PA + $100-200 artist fees + $100 promotional materials = ~$800-1,100 total. Can be folded into the crowdfund goal.

---

### Option B — DevCon Booth / Demo Table

If a side event is too complex to organize, an alternative is a demo table at DevCon itself or at a major side event hub (e.g., ETHIndia fringe, Starkware satellite).

**Format:** Table with a laptop showing WaveWarZ live, Zaal and Hurricane available for 1-on-1 demos. Run a micro-battle with anyone who stops by.

**Cost:** Minimal — just the demo setup. No venue needed.

**Downside:** Passive. Does not generate the viral social moment that a full battle event does.

---

### Option C — Sponsor a DevCon Side Stage

Some DevCon community events have music moments built in (hackathon kickoffs, after-parties). ZAO could approach organizers of an existing event to add a WaveWarZ battle moment.

**How:** Reach out via the DevCon Discord/forum in September to events seeking musicians or entertainment. Offer: "Free WaveWarZ battle moment (30 minutes) — we bring the tech, you bring the crowd."

**Cost:** Minimal. ZAO provides the battle infrastructure; the host event provides venue.

## WaveWarZ Battle Adaptation for Live IRL Format

Running WaveWarZ battles live requires a few adaptations from the normal online format:

**Voting mechanism:**
- Option A (recommended): Audience votes via QR code linking to a WaveWarZ voting page — live tally shown on projector screen. Uses existing WaveWarZ vote infrastructure.
- Option B: Hands-up audience vote (no-tech fallback if wifi is unreliable)
- Option C: Farcaster cast poll (voting via FC clients — requires audience to be on Farcaster)

**Audio setup:**
- Need a PA system capable of playing digital audio at venue-quality volume
- Two tracks play back-to-back (no DJ mixing needed)
- Hurricane handles tech; Zaal hosts/MCs

**Battle host script (5-minute format):**
1. "Two tracks. One winner. The crowd decides. No algorithm. No payola."
2. Play Track A (60-90 sec) → brief pause → Play Track B (60-90 sec)
3. "Pull out your phone and scan the QR code — vote for A or B"
4. 2-minute live voting window → results on screen
5. Announce winner: "Track [A/B] wins. [ARTIST NAME] earned SOL regardless."
