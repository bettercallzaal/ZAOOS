## How to Close a Hesitant Sponsor

### If they say "what's the ROI?"

```
The ZAO community is 188+ members — music artists, creators, and web3-native fans who are early adopters in tech-forward businesses. ZAOstock is their first live IRL event. You're not buying banner space; you're buying a moment in a story.

Two data points that are measurable:
- We'll post about you 3 times to our combined X + Farcaster + Telegram audience (estimated reach: [N] impressions)
- Your name will be in the Mirror article we publish post-ZAOstock (citable, permanent, SEO-indexed)

What kind of ROI would make this a yes for you?
```

### If they say "I need more time to decide"

```
Understood. I'm locking in sponsors by September 1 — after that, the program is printed and the backdrop goes to the designer. I'd love to hold a slot for you. Is there anything that would help you decide faster?
```

### If they say "can I pay less?"

For local Maine businesses at the $500 or $1,000 tier:
- Offer an in-kind sponsorship trade: if they can provide $500 equivalent in product or service (catering, photography, gear), ZOE treats them as a confirmed Community sponsor.
- Examples: food/beverage sponsorship (local restaurant), photography sponsorship (local photographer), AV equipment (local AV company).

### If they say "I've never heard of ZAO"

Send them doc 1651 (ZAO DAO Case Study) link or paste:
```
ZAO has run 100+ consecutive weekly governance sessions on Optimism Mainnet. WaveWarZ has settled 1,245 music battles with $524 SOL in fan trading volume and automatic artist payouts. ZAOstock is the first IRL event. We're bringing what we've built online to a live stage in Ellsworth, Maine.

Here's the case study: [ZAOOS doc 1651 link or zaoos.com/faq]
```

## Sponsor Supabase Schema

```sql
CREATE TABLE zaostock_sponsors (
  id              uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  company_name    text NOT NULL,
  contact_name    text,
  contact_email   text,
  tier            text CHECK (tier IN ('artist', 'community', 'stage', 'title')),
  amount_usd      integer,
  status          text DEFAULT 'pitched' CHECK (status IN ('pitched', 'interested', 'confirmed', 'paid', 'declined')),
  logo_url        text,
  payment_method  text,
  promo_code      text,
  has_banner      boolean DEFAULT false,
  has_speaker     boolean DEFAULT false,
  speaker_bio     text,
  notes           text,
  created_at      timestamptz DEFAULT now(),
  updated_at      timestamptz DEFAULT now()
);
```

ZOE upsert pattern (on sponsor status change):
```typescript
const { error } = await supabase
  .from('zaostock_sponsors')
  .upsert({
    company_name: sponsorName,
    status: newStatus,
    updated_at: new Date().toISOString(),
  }, { onConflict: 'company_name' })

if (error) {
  await notifyZaalTelegram(`⚠️ Sponsor update failed for ${sponsorName}: ${error.message}`)
}
```
