## Thread 1 - The pre-record + live-Q&A format

The core idea, credited to yerbearzerker (Jordan): run the learning before the hackathon, Farcaster-agentic-workshop style. Zaal's framing: a lot of tool-builders have great tools but people don't know how to use them best - so the talks teach the tools, with the creators present.

For musicians specifically, the live-presentation part is the friction. Solution: the duo pre-records a tight 15-30 min presentation (scripted, multiple takes), and all three watch it together live. The artists don't perform live - they just answer questions at the end. Clementine/Santiago liked this: "having the presentation ready is going to give more freedom for timing to let people get confidence" - the value is in the feedback and the room's questions, not the delivery.
