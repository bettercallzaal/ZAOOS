## What was covered (the Discord bot setup flow)

Zaal walked Iman through the Discord developer portal step by step:
1. Go to the portal -> **Installation**.
2. Grab the **Client ID** and the **Client Secret** (the secret goes in the `.env` file - do not expose it; set it later).
3. The **OAuth URL** is generated by the portal.
4. Go to the **Bot** section -> scroll down -> enable the three **privileged gateway intents**: **Presence Intent**, **Server Members Intent**, and **Message Content Intent**.
5. Back to **Installation** -> bottom -> copy the **Discord-provided install link** -> paste into the URL bar -> add the bot to the server.

Zaal: he will handle the server-hosting + environment-variable part later; once Iman adds the bot to the server they are good to go. The same flow then repeats for whatever other servers Iman needs it on.
