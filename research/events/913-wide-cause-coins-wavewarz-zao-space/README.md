## Summary
Community deep-dive on **Wide (Wyoming Decentralized Exchange)**, founded Oct 2025 by Aaron Rafferty. Wide launches cause-specific crypto tokens (first: **EAT** for hunger, with Feed the Children as exclusive 50% partner) where **trading fees perpetually fund nonprofits** (~3-10 meals per $1). A debit card via I3 Bank will route interchange fees to causes (the "Tom's Shoes model made scalable"). The back half became a real **Wide x WaveWarZ x ZAO** crossover: WaveWarZ's 1%-to-artist + benefit-battle model and The ZAO's soulbound RESPECT tokens mapped against Wide's attention->impact flywheel. Zaal drove most of the mechanics Q&A.

## ZAO relevance (the gold)
- **Same flywheel:** Wide's cause-coin (trade -> fee -> nonprofit) is structurally the same attention->impact loop as WaveWarZ benefit battles (1% of every trade -> artist/cause, e.g. the PolyRaiders water-pump battle).
- **Partnership surface:** Aaron + Candy + Zaal openly explored Wide x WaveWarZ x ZAO. Cleanest fit = WaveWarZ benefit battles as a "cause-coin" expression; ZAO as the artist-DAO layer.
- The ZAO's soulbound RESPECT tokens (illiquid, contribution-earned) came up as the governance contrast to liquid cause-coins.
- Farcaster repeatedly named as the better home for these cause-Spaces (recording/audiogram tooling).
- Moses / PolyRaiders (water pump in Nigeria) - existing ZAO-adjacent impact builder ([[project_moses_polyraiders]], doc 884).
