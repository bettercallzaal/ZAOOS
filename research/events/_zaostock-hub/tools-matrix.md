## Section 6: Composability Map (Tools That Chain Together)

**Recommended integration sequences for max value:**

### Sponsor Pipeline (Docs 01, 08)
```
1. Clay.com [enrich prospects]
   ↓

   ↓

   ↓
4. Claude [Bayesian revenue forecast]
   ↓
5. Slack [weekly variance alerts]
```
**Cost:** $200 Clay + $0 Claude (existing key) = $200/mo.

### Artist Discovery (Docs 02)
```
1. Neynar API [find Farcaster artists]
   ↓
2. Spotify API [check releases + streams]
   ↓
3. Claude [score vetting rubric]
   ↓
4. Claude [draft outreach emails]
   ↓
5. Farcaster SDK [auto-post lineup when confirmed]
```
**Cost:** $0 (all free APIs + existing Claude key).

### Volunteer Workflow (Docs 03)
```
1. Google Forms [collect applications]
   ↓
2. Claude [extract skills from bios]
   ↓
3. Claude [suggest role + starter tasks]
   ↓
4. SignUp.com [schedule shifts (auto-generated from OR-Tools)]
   ↓
5. Twilio SMS [send 3 reminders (Wed, Fri, morning-of)]
   ↓
6. Typeform [post-event NPS + thank-you]
```
**Cost:** $99 SignUp + $0.45 Twilio (60 SMS) = $99.45 total.

### Meeting Operations (Docs 12, 13)
```
1. Google Meet [record to Drive]
   ↓
2. Deepgram [async transcription]
   ↓
3. Claude [extract summary + decisions + actions]
   ↓
4. Supabase [store in stock_action_items]
   ↓
5. ZOE/Telegram [send owner reminders at -48h, -24h]
   ↓
6. Doc 13 [log all decisions to decision_log table]
```
**Cost:** $1.50 Deepgram per meeting.

### Social + Media (Docs 06, 09)
```
1. S3 [store all photos + videos]
   ↓
2. Claude vision [batch score photo shareability + auto-caption]
   ↓
3. Runway Gen-3 [create 30-sec reels from clips]
   ↓
4. Firefly [cross-post to Farcaster + X + Bluesky]
   ↓
5. Paragraph [email highlights to alumni list]
```
**Cost:** $0.15 Claude vision + $30-50 Runway + $0 Firefly = $30-50 one-time.

### Post-Event Analytics (Docs 07, 10)
```
1. Typeform [collect NPS surveys]
   ↓
2. Claude [sentiment + theme extraction]
   ↓
3. Neynar + X API [scrape social mentions]
   ↓
4. Claude [synthesize sponsor ROI report]
   ↓
5. Puppeteer [generate PDF for each sponsor]
```
**Cost:** $0 (free APIs + Claude batching).
