1. ZOE pre-drafts all Sep 22-26 and Oct 3 casts by Sep 15 (as templates with [BRACKET] placeholders)
2. ZOE fills brackets on event day using WW API (wavewarz.info/api/public/stats)
3. Battle result casts fire automatically after battle closes on WW platform (event webhook from doc 1615)

**Approval flow:** Pre-drafted templates approved by Zaal in a single review session Sep 14. Event-day fills are auto-posted without approval for standard battle results. Novel casts (unexpected moments, big milestones) go to Zaal for 5-min approval before posting.
