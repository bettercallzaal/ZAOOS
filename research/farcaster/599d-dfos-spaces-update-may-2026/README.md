## DFOS State (March 2026 → May 2026)

Source: blog.metalabel.com/what-dfos-is-becoming/ (March 2026 post; pulled 2026-05-03).

### Core Pitch

> "a platform where every community designs its own internet"

### Features Shipping

| Feature | Detail |
|---------|--------|
| Subgroups | Members subscribe selectively. 500-person communities feel intimate via "your corner of it." |
| Customizable homepages | Tiles, feeds, pinned posts, embedded apps. "A music space looks like a music space." |
| Attention redesign | 20x notification reduction; daily/weekly digests for non-urgent updates. |

### Access

Invite-based alpha. Founding community members only. No pricing disclosed.

## Convergence with ZAOOS

| Dimension | DFOS | ZAOOS |
|-----------|------|-------|
| Pitch | Each community designs its internet | Each project incubates in the lab; graduates to own internet |
| Customization | Per-space homepage + embedded apps | community.config.ts + per-feature graduation |
| Notification | 20x reduction via digests | iron-session + push notif backbone (varies) |
| Onchain | None visible | Wagmi + Supabase + 0xSplits + ZABAL holdings |
| Agents | None | ZOE + Hermes + ZOEY + WALLET + ROLO |
| Distribution | Private alpha | Open source, Farcaster-native |
| Audio/spaces | None visible | Spaces + Stream.io |

ZAOOS overlaps with DFOS only on the "custom community internet" framing. ZAOOS also covers governance, agents, music, distribution. DFOS is narrower and more polished.

## Differentiation Strategy

1. **Lead with onchain** - DFOS has no token, no governance primitive. ZAO does (ZABAL, Respect, ZOLs).
2. **Lead with agents** - DFOS spaces are static. ZAO spaces have ZOE + Hermes + agent squad.
3. **Lead with music** - DFOS is multi-vertical. ZAO is music-and-art-first.
4. **Open source** - DFOS is closed alpha. ZAOOS code is in this repo.
