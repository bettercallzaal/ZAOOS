## The Hierarchy of Truth (Read Before Using)

1. ZAO Fractal has run **100+ consecutive weeks** — that number is the anchor stat. Use it.
2. ZOR can't be bought — **earned by showing up and contributing**. Distinguish from tokens.
3. Lead with **community, not governance tooling**. Belonging-first.
4. Audience-appropriate: musicians hear "recognition," builders hear "on-chain governance," press hears "longest-running."
