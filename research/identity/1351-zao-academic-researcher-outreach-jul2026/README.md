## Part 4: Making ZAOOS More Citable

Academic researchers cite sources in specific ways. ZAOOS docs can become more citable with minor additions:

| Improvement | How | Impact |
|-------------|-----|--------|
| Add DOI-like permanent links | Archive each doc on Arweave — the Arweave URL is permanent | Researchers can cite a URL that won't 404 |
| Add creation dates (already done) | All docs have `created:` frontmatter | Enables correct citation date |
| Add author field | Add `author: Zaal Panthaki / ZAO research team` to frontmatter | Enables author credit in citations |
| Add ZAOOS version number | Tag releases (v1.0, v2.0) when crossing doc milestones | Enables citing "ZAOOS v2.0" not just a URL |
| Add data licensing statement | Add MIT or CC-BY license declaration to root README | Enables researchers to use data without legal ambiguity |

**Priority:** Add a license declaration to ZAOOS root README so researchers know they can freely cite and build on the data.
