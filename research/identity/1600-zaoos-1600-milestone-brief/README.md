## What ZAOOS Is

ZAOOS (ZAO Operating System) is ZAO's public research archive — a CC-BY licensed corpus of documents covering every aspect of ZAO's governance, music platform, events, community, and operations. It runs on GitHub as a public repository with an automated PR-and-merge pipeline.

**Repository:** github.com/bettercallzaal/ZAOOS  
**License:** CC-BY (Creative Commons Attribution) — all documents are free to use, cite, fork, and build on  
**Document format:** Markdown, one README.md per document directory  
**Pipeline:** Every doc is a git branch → PR → auto-merge to main

**What ZAOOS documents:**
- WaveWarZ platform operations (battles, payouts, artist tracking)
- ZAO governance (Fractal Democracy sessions, OREC submissions, ZOR token mechanics)
- Events (COC Concertz, Africa Battle Week, ZAOstock, ZAOville)
- Community (Telegram ops, artist activation, spotlights, ZABAL accelerator)
- Identity (press kits, GEO entity briefs, citable claims, media pitches)
- Business (grant applications, revenue models, partner activations)
- Technology (ZOE automation, Hurricane dev, API specs, Mini App)
- Farcaster (channel strategy, ZOL, Neynar integration)
