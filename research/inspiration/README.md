# Inspiration Library

> Auto-generated daily by the "Steal Like an Artist" scheduled agent. Each entry analyzes UX patterns from music, social, and creative apps — then maps them to ZAO OS through the lens of becoming a **decentralized music label**.

## How This Works

A Claude Code scheduled agent runs daily at 10am EDT:
1. Picks 3 apps from a rotating pool (Spotify, Bandcamp, SoundCloud, Discord, Are.na, Tidal, Apple Music, Audius, Sound.xyz, Catalog, Zora, Lens, Paragraph, Mirror, Substack)
2. Analyzes one specific UX pattern from each
3. Cross-references against existing research docs (243+)
4. Frames every insight through: "How does this get ZAO closer to being a decentralized music label?"
5. Saves to `research/inspiration/YYYY-MM-DD.md`

## Decentralized Music Label North Star

Everything in this folder is filtered through one question:
> **What would a community-owned, on-chain music label look like — and how does ZAO OS become the operating system for it?**

The five pillars:
1. **Artist development** — discovery, curation, mentorship
2. **Distribution** — multi-platform publishing, on-chain releases
3. **Revenue** — splits, royalties, treasury, community funding
4. **Governance** — who decides what gets released, promoted, funded
5. **Culture** — taste-making, community identity, brand
