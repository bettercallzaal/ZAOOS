# 1341 — WaveWarZ MAIN Event Strategy (Jul 2026)

> **What is a MAIN Event?** A MAIN Event is a scheduled, promoted, multi-round WaveWarZ battle session. Unlike Quick Battles (ad-hoc, short), MAIN Events follow a fixed cadence (historically Sunday 8pm EST), feature multiple battle rounds, draw larger audiences, and generate significantly more SOL volume per session.
>
> **Why this matters:** 50 MAIN events (162 battles) have collectively generated the majority of WaveWarZ's 523.991 SOL total volume. Scaling MAIN events is the single highest-leverage action for WaveWarZ revenue, artist payouts, and ZAOstock pipeline.

### MAIN Event Cadence

**Average sessions per MAIN event:** 162 battles / 50 events = **3.24 battles per event**

### MAIN Event → ZAOstock Pipeline

MAIN event participants are the **natural ZAOstock audience**:
- They've already traded SOL in battles → proven engagement
- Top MAIN event performers → candidate artists for ZAOstock lineup
- Regular MAIN event traders → newsletter subscribers, ticket buyers

**Action:** After each MAIN event, send participants an invite to ZAOstock (newsletter signup CTA or direct DM).

### Artist Payout Multiplication

Every additional MAIN event = more battle rounds = more 1% payouts to artists.
- Current: 9.0988 SOL total artist payouts / 50 MAIN events = **0.18 SOL per MAIN event** (avg artist payout contribution)
- If MAIN events double to 100 → projected artist payouts: ~18 SOL (~$1,350 at Jul 2026)
- This is the "artists paid first" North Star (#2) in action

## Part 4: How to Run More MAIN Events

### Current Bottleneck

MAIN events require:
1. Hurricane to schedule/announce (core platform team)
2. Willing artists on both sides
3. Community awareness to drive bets

The whitepaper noted Sunday 8pm EST cadence but not every week has had a MAIN event (50 events / ~39 weeks = not weekly).

### Scale Levers

**Lever 1: ZAO-hosted MAIN Events** (available NOW)
- ZAO and Hurricane co-host a "ZAO MAIN Event" monthly as a regular calendar item
- ZAO selects the artists, promotes through ZAO channels (@bettercallzaal, /zao, newsletter)
- This adds ~1 MAIN event/month = +12/year on top of existing cadence

**Lever 2: ZABAL Games Integration** (Aug-Nov 2026)
- ZABAL August Finals (closes Aug 18) produces a champion → that champion gets a MAIN event slot
- ZABAL Season 2 (Sep-Nov) → each monthly finalist earns a MAIN event slot
- Expected: 3-4 additional MAIN events from ZABAL pipeline

**Lever 3: Sponsor-Backed MAIN Events** (after sponsor deck, doc 1334)
- Local Maine sponsors (Ellsworth American, Bangor Savings) can sponsor a named MAIN event: "Bangor Savings Community Battle"
- Adds 1-2 sponsor MAIN events per season

**Lever 4: ZAOstock Qualifier Series** (Aug 1 – Sep 30)
- 8 qualifier MAIN events (one per artist slot), results determine ZAOstock lineup
- This creates a built-in reason for 8 additional MAIN events before Oct 3
- Dramatic stakes = higher SOL, higher audience, better press angle

**Lever 5: Partner-Led MAIN Events** (Q4 2026)
- Juke, Magnetiq, Empire Builder, RAM SongChain each host one MAIN event per quarter featuring their artist roster
- Expected: 4 partner MAIN events per quarter = +16/year
