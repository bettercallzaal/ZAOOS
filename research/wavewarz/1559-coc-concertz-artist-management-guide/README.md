**Type:** OPERATIONS-GUIDE

## COC Concertz Artist Flow

Each COC Concertz show features 4–6 artists (2 in the MAIN battle + 2–4 supporting performers). The MAIN battle artists are selected via ZOR holder governance vote. Supporting performers are curated by Zaal from the WaveWarZ top artist list.

**Artist pipeline per show:**

```
ZOR holder vote (artist nomination) → Zaal DM invitation → artist confirms
→ sound check brief → ZOE pre-show posts → show day → ZOE post-show DMs
→ ZAOstock invitation (if strong performance)
```

---

## Phase 1: MAIN Battle Artist Selection

**Timeline:** 2 Thursdays before show date

**Process:**
1. ZOR holders nominate MAIN battle candidates in Thursday governance session
2. Zaal facilitates vote using Fractal Democracy (highest-Respect voters' choices prioritized)
3. OREC records the vote on Optimism Mainnet (doc 1540 archive template)
4. Top 2 nominees = MAIN battle artists for that show

**Selection criteria (ZOR holder guidance):**
- Active WaveWarZ battle history (preference for artists with ≥3 battles)
- No repeat MAIN appearance within 2 consecutive shows (keep shows fresh)
- At least 1 artist per show should be a COC or ZAOstock newcomer (expand the ecosystem)
- Africa Battle Week (Sep 26): prioritize African artist in MAIN battle if Africa partner confirmed

After ZOR vote: Zaal sends invitation within 24 hours.

### Template: MAIN Battle Invite
```
Hey [Name] 👋

The show format:
• MAIN battle: you vs. [Artist B], 15-min live audience vote
• You both get paid — loser earns guaranteed payout
• ZOR holders vote from the crowd (and remote)
• ZOE posts the result live on @wavewarz and Telegram

What you need: 1 track on Audius/SoundCloud + Phantom wallet for payout.
Show time: [TIME] EST. Sound check: [TIME] EST.

RSVP here (so I can mark you confirmed): reply YES or DM me on [X/Telegram]

— Zaal
```

### Template: Supporting Performer Invite
```
Hey [Name] 👋

We'd love to have you perform at COC Concertz #[N] — [DATE].

Supporting set: 15–20 min (2-3 tracks) before the MAIN battle.
```

## Phase 4: Show Day

**ZOE tasks (day-of):**
- 2 hours before: post @wavewarz X preview (artists + battle matchup)
- 30 min before: post Telegram reminder to ZAO community
- During battle: live-post vote count (if API integration live) OR post results within 2 min of close
- After battle: post result with payout amounts + Solana tx hash

**Zaal tasks:**
- Introduce each artist on mic (use bio from artist DM exchange)
- Open the WaveWarZ battle on-stage (navigate wavewarz.info/battle/create or coordinate with Hurricane)
- Announce result live: "[Artist A] wins! [Artist B] — the loser still got paid: [SOL amount] SOL"
