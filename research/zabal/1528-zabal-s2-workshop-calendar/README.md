## ZAOOS Contribution Requirements

Each participant must contribute to ZAOOS during S2:
- **Minimum:** 2 docs over 12 weeks
- **Recommended:** 1 doc per 2 weeks (6 total)
- **Doc types:** research, how-to, case study, reflection, or technical reference
- **ZOL integration:** ZOL bot indexes all ZAOOS contributions and surfaces them in cohort check-ins

ZOE tracks ZAOOS submissions weekly and surfaces to Zaal in EOD report.
